# Données

Les fichiers sources sont conservés dans `raw/`. Les résultats reproductibles
peuvent être écrits dans `processed/` ; aucun résultat fabriqué n'y est livré.

| Fichier | Description | Origine | Sensibilité |
|---|---|---|---|
| `iris.csv` | Mesures morphologiques de 150 iris | Jeu Iris classique, copie du dépôt d'origine | Aucune donnée personnelle |
| `tips.csv` | Pourboires et caractéristiques de repas | Jeu d'exemple pédagogique, copie du dépôt d'origine | Aucune donnée personnelle directe |
| `spx.csv` | Série historique de l'indice S&P 500 | Copie du dépôt d'origine ; provenance primaire non documentée | Aucune donnée personnelle |

Les fichiers olympiques historiques (plus de 40 Mo) et le classeur dupliquant ces
données sont conservés dans la sauvegarde locale `archive_originale/`, ignorée par
Git. Leur licence et leur provenance doivent être confirmées avant republication.
